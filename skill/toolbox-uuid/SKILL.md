## What I do

Run the following command to generate a UUID:

```
./skill/toolbox-uuid/get-uuid
```

## When to use me

Use me when you need to generate a UUID for any purpose, such as unique identifiers
for database entries, session tokens, or any scenario requiring a unique reference.
